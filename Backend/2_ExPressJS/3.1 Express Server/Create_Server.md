# Creating an Express Server

- Create Directory
- Create index.js file
- Initialise NPM
- Install the Express package
- Write Server Application in index.js
- Start Server

## Start a Server

- Open `Terminal`
- Change the Directory using `cd <directory path>`
- `node .\index.js` to start the server
